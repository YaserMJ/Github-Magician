# Github Magician
> _Yaser Muhammed Jaser Saleh_


## **Description :**
A React & Github based application that searches for users,popular repositories sorted by programming languages and battling between Github accounts!
#### Technologies used : 
1. React.
2. React-Router.
3. Bootstrap.
4. Github API.
5. Axios.
6. SASS and others.
